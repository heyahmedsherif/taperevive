# TapeRevive - Coming Soon Page

Simple landing page for TapeRevive.com

## About

TapeRevive helps people convert their precious VHS home videos to digital format, preserving memories before they're lost forever.

## Deployment

This site is hosted on GitHub Pages and uses a custom domain (taperevive.com).

## Local Development

Simply open `index.html` in your browser to preview the page.

## Technologies

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript
- Google Fonts (Inter)
